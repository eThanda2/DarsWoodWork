# Dara Woodshop 🪚

Welcome to the official source code for **Dara Woodshop**, a premium, bilingual web portfolio showcasing custom, handcrafted woodworking projects. 

## 🌐 Live Website
**[https://darawoodshop.com](https://darawoodshop.com)**

## ✨ Features
- **Bilingual Interface:** Seamless side-by-side English & Arabic language switching with full RTL (Right-to-Left) layout support.
- **Premium Design:** A modern, luxurious aesthetic featuring glassmorphism elements, a sticky dark-green navigation bar, and rich wood tones.
- **Responsive Layout:** Perfectly optimized for all devices, from ultra-wide desktop monitors to mobile phones.
- **Interactive Gallery:** Embedded YouTube videos highlighting recent builds directly within a sleek, custom modal.
- **"Get in Touch" Module:** A beautifully designed contact form paired with quick-access social media links.

## 🛠️ Tech Stack
- **HTML5:** Semantic, accessible markup.
- **Vanilla CSS3:** Custom styling utilizing CSS variables for consistent theming and ultra-fast load times. No heavy frameworks!
- **Vanilla JavaScript:** Lightweight DOM manipulation for language toggling, modal control, and smooth scrolling.
- **Hosting:** GitHub Pages with a custom domain and automated HTTPS via Let's Encrypt.

## 🚀 Local Development
To run this project locally on your machine:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No build steps or complex server setups are required!
